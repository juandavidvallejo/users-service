# users-service

install node

npm install

install mongodb

1. node app.sj (http://localhost:3001/api/users)
2. Use a Postman to send a POST with this JSON:
   {

  "first_name": "Carlos",
  "last_name": "Parra",
  "address": "Calle 93 No 11 34",
  "city":  "Bogota",
  "neighborhood":  "Chico",
  "birthdate":  "03.12.1990"
  }
